THE INTEGRAL CALCULUS

 Basic Integration

The integral calculus is concerned with the inverse problem: if the rate of change of a function is known, what is the function itself? In other words, a function y has to be found when the derivative dy/dx is given. 

Integration is just the direct opposite of differentiation. It "works back" the derivative of a function to the root function itself. We will discuss basic integration and the tools you would need to evaluate different kinds of integrals.

Example 3: Given that f'(x)=3x2 - 6x + 1 and f (3) = 5, find the original function f(x).

Solution

f'(x) is the derivative of the function. Given f'(x), we integrate this term to determine f(x). 

Since we know the value of this function, f(x) for x=3, we can also determine the value of the constant C usually included after integrating a function. 

So, effectively, we will be able to write the function completely for x.

int f'(x) = 3x3/3 - 6x2/2 + x= x3 - 3x2 + x + C

ie f(x) = x3 - 3x2 + x + C 

:. f(3) = 33- 3(3)2 + 3 + C = 5 => C = 5 – 3 = 2

:. f(x) = x3 - 3x2 + x + 2

Example 4: Evaluate int [3x2 - 2x - 12]3-2 dx

Solution

This is an example of the definite integral, "definite" because we are going to evaluate this integral numerically for the above limits. 

The upper limit, (superscript) is 3 while the lower limit, or subscript is -2. The two numbers represent the limit of the definite integral.

This is similar to the how we evaluate the derivative of a function at a given point, say x= -3, written dy/dx|x = -3.

It's also worth noting that the value of the definite integral represents the area under the graph of the function and the x-axis for the chosen limits. 

This means that the above question could be rephrased thus: " What is the area b/w the curve of 3x2-2x-12 and the x-axis for the interval [-2,3]?".

When we proceed,

int (3x2 - 2x - 12) = 3x3/3 - 2x2/2 -12x = x3 - x2 - 12x 

To evaluate the indefinite integral for the limits, we substitute x for -2 and 3 and then subtract the lower limit from the upper limit.

UL: 33-32-12(3) = -18

LW: (-2)3 - (-2)2 - 12(-2) = -8 – 4 + 24 = 12

:. [3x2-2x-12]3-2 dx, the area b/w the curve of 3x2 - 2x - 12 and the x-axis for the interval [-2, 3] = -18 - 12 = -30 sq. units
