Equality of Matrices

Two matrices A and B are equal if and only if they have the same elements in the same positions (i.e. they are the same in all aspects.)

Matrix Multiplication 

To be able to determine the product of a matrix A and a second matrix B, the number of columns in A must be equal to the number of rows in B.

Matrix multiplication is a row-column operation. Each row in the first matrix successively multiplies each column in the second matrix and the results are added.

Determinant of a Square Matrix

A square matrix is a matrix with the same number of rows and columns eg 3×3, 4×4, 2×2 matrices etc.

In general, a n × n matrix is called a matrix of order n eg 3 × 3 matrix is a matrix of order 3, 2 × 2 is order 2 and so on. 

Square matrices have a unique property–determinant! This means a matrix cannot have a determinant unless it is a square matrix. 

The determinant of a square matrix, written det A or |A| (NOT absolute value) is defined as follows:

For a 2 × 2 matrix

Det A = ad - bc

For a 3 × 3 matrix, 

A =  

|A| = a (ei - hf) - b (id - gf) + c (dj - ge)

this means that to determine the expressions in 3 × 3, eliminate the row and column that the preceding letter belongs to and use the formula for the determinant of a 2×2 matrix.

Always notice the - minus sign in front of b (id - gf)

Solution

This is a square matrix of order 3. Applying this |A| = a (ei - hf) - b (id - gf) + c (dj - ge)

The determinant is found as follows:

Det = 1 (2 × 1 -1 × 0) - 0 (2 × 1 – 0 × 0) + 1 (1 × 1- 0 × 1)

Det = 1 (2 - 0) - 0 (2 - 0) + 1 (1 - 0)

Det = 2 - 0 + 1

: . Det = 3
